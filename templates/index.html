markdowner

a pastebin for markdown that renders it into html fragments.

uploading

  • you can upload through the web interface, linked in the header
  • cURL can also be used, for example "curl -H "Content-Type: text/markdown" -X POST --data-binary '@test-file.md' {{ url_for('upload', _external=True) }}". it is is important to use "--data-binary", as otherwise newlines will be mangled.

downloading

  • if you wish to get the raw data without the header, or without the dodgy CSS, you can access it at {{ url_for('rawFetch', pasteID = "fakeID",_external=True) }}
  • this is useful if you want to download the rendered HTML with cURL, as in "curl {{ url_for('rawFetch', pasteID = "fakeID",_external=True) }} > file.html"

notes

  • this uses the "python-markdown" module to handle the rendering, so don't get annoyed at me when it doesn't work
  • in some sort of attempt to prevent XSS, all html in the markdown is escaped before it is rendered - sorry if you need that
